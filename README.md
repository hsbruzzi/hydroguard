# HydroGuard Avellaneda

Proyecto base integrado:
- `backend/` FastAPI
- `frontend/` Flutter

## Flujo recomendado
1. Levantar backend en `http://localhost:8000`
2. Ejecutar app Flutter
3. La app consulta `GET /estado`
